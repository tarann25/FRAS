PROJECT OBJECTIVE                                                                                                                                                         
   Build a real-time face recognition attendance system that:                                                                                                                
   - Registers users with face data                                                                                                                                          
   - Recognizes faces via webcam                                                                                                                                             
   - Marks attendance automatically                                                                                                                                          
   - Stores records in a database                                                                                                                                            
   - Provides a web dashboard                                                                                                                                                
                                                                                                                                                                             
   TECH STACK                                                                                                                                                                
   - Python 3.10+                                                                                                                                                            
   - Flask (backend)                                                                                                                                                         
   - OpenCV (video capture)                                                                                                                                                  
   - face_recognition (face encoding + matching)                                                                                                                             
   - NumPy (vector operations)                                                                                                                                               
   - Pandas (attendance logs)                                                                                                                                                
   - SQLite (database)                                                                                                                                                       
   - HTML/CSS/JS (frontend)

   PHASE 2 — FACE REGISTRATION MODULE                                                                                                                                        
   ────────────────────────────────────                                                                                                                                      
                                                                                                                                                                             
   OBJECTIVE:                                                                                                                                                                
   Allow admin to register a new user and store their face encoding.                                                                                                         
                                                                                                                                                                             
   STEP 2.1 — IMAGE CAPTURE                                                                                                                                                  
                                                                                                                                                                             
   Implement function:                                                                                                                                                       
   capture_user_images(user_id, num_samples=10)                                                                                                                              
                                                                                                                                                                             
   Requirements:                                                                                                                                                             
   - Capture frames from webcam                                                                                                                                              
   - Detect face                                                                                                                                                             
   - Save cropped face images to dataset/<user_id>/                                                                                                                          
   - Capture multiple samples (~10)                                                                                                                                          
                                                                                                                                                                             
   STEP 2.2 — GENERATE FACE ENCODINGS                                                                                                                                        
                                                                                                                                                                             
   Implement function:                                                                                                                                                       
   encode_user_faces()                                                                                                                                                       
                                                                                                                                                                             
   Logic:                                                                                                                                                                    
   For each image in dataset:                                                                                                                                                
   1. Load image                                                                                                                                                             
   2. Convert BGR → RGB                                                                                                                                                      
   3. Detect face                                                                                                                                                            
   4. Generate 128-d encoding                                                                                                                                                
   5. Store in dictionary:                                                                                                                                                   
      { "user_id": encoding_vector }                                                                                                                                         
   6. Save to encodings/encodings.pkl using pickle                                                                                                                           
                                                                                                                                                                             
   Exit condition:                                                                                                                                                           
   Encodings file exists and loads correctly.                                                                                                                                
                                                                                                                                                                             
   ────────────────────────────────────                                                                                                                                      
   PHASE 3 — REAL-TIME FACE RECOGNITION                                                                                                                                      
   ────────────────────────────────────                                                                                                                                      
                                                                                                                                                                             
   Core function:                                                                                                                                                            
   recognize_faces_realtime()                                                                                                                                                
                                                                                                                                                                             
   Required pipeline per frame:                                                                                                                                              
   1. Capture frame via OpenCV                                                                                                                                               
   2. Resize frame (0.25 scale)                                                                                                                                              
   3. Convert BGR → RGB                                                                                                                                                      
   4. Detect face locations                                                                                                                                                  
   5. Generate encodings                                                                                                                                                     
   6. Compare with known encodings                                                                                                                                           
   7. Find best match                                                                                                                                                        
   8. Draw bounding box + name                                                                                                                                               
   9. Return recognized user_id                                                                                                                                              
                                                                                                                                                                             
   MATCHING LOGIC (CRITICAL)                                                                                                                                                 
                                                                                                                                                                             
   Must use:                                                                                                                                                                 
   - face_recognition.compare_faces()                                                                                                                                        
   - face_recognition.face_distance()                                                                                                                                        
                                                                                                                                                                             
   Decision rule:                                                                                                                                                            
   - Pick minimum distance                                                                                                                                                   
   - Accept match if distance < 0.6                                                                                                                                          
   - Else mark as "Unknown"
